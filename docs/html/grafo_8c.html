Detailed Description

Implementação das operações para gestão de grafos de antenas.

Implementa todas as funções declaradas em grafo.h, incluindo:

  • Criação e destruição de grafos
  • Operações básicas com vértices e arestas
  • Algoritmos de travessia (DFS e BFS)
  • Cálculo de caminhos entre vértices
  • Deteção de intersecções entre frequências

int adicionar_aresta ( Vertice * origem,
Vertice * destino )

Adiciona uma aresta entre dois vértices (antenas da mesma frequência)

Parameters
origemVértice de origem
destinoVértice de destino
Returns
0 em caso de sucesso, -1 em caso de erro, -2 se frequências diferentes

A aresta é bidirecional (grafo não direcionado) e não são permitidas:

  • Arestas entre vértices com frequências diferentes
  • Arestas duplicadas entre o mesmo par de vértices
